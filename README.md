# 🤖 Telegram LeadGen & Broadcast Bot

Профессиональный инструмент для сбора лидов, выдачи лид-магнитов и проведения маркетинговых рассылок. Бот полностью автономен и предоставляет админ-панель для управления базой данных.

## 🚀 Ключевой функционал
- **Smart Onboarding:** Автоматический сбор контактов (Имя, Телефон, Email) с валидацией данных.
- **Lead Magnet:** Мгновенная выдача PDF-материалов после успешной регистрации.
- **Deep Linking:** Отслеживание источника трафика (Instagram, VK, Direct) через стартовые параметры.
- **Broadcast Engine:** Встроенный конструктор рассылок (текст + фото/видео) с поддержкой кнопок-ссылок.
- **Anti-Flood System:** Безопасная отправка массовых сообщений с защитой от блокировок Telegram API.

## 📊 Управление данными
- **Local Persistence:** Хранение всех данных в реляционной базе **SQLite**.
- **Cloud Sync:** Автоматическое дублирование лидов в Excel-отчет на **Yandex Disk** в реальном времени.
- **Admin Dashboard:** Скрытое меню для владельца с возможностью мгновенной выгрузки всей БД в `.xlsx`.
- **System Logging:** Полный аудит активности и ошибок через библиотеку **Winston**.

## 🛠 Технологический стек
- **Runtime:** Node.js (v20+)
- **Framework:** Telegraf.js (Telegram Bot API)
- **Database:** SQLite3
- **Storage Integrations:** Yandex Disk API (REST via Axios)
- **Reporting:** ExcelJS
- **Logger:** Winston

## 📦 Установка и запуск

1. Клонировать репозиторий.
2. Выполнить `npm install`.
3. Создать файл `.env` (см. раздел Безопасность) и добавить токен бота.
4. Запустить: `node index.js`.
