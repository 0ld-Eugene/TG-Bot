# Telegram Lead Generation Bot

Бот для автоматизации сбора лидов и выдачи полезных материалов (lead magnet).

## 🚀 Функционал
- Анкетирование пользователя (Имя, Телефон, Email).
- Сохранение данных в локальную БД **SQLite**.
- Дублирование данных в **Google Таблицы** в реальном времени.
- Админ-панель с выгрузкой отчета в **Excel**.

## 🛠 Технологии
- **Backend:** Node.js, Telegraf
- **Database:** SQLite3
- **Integrations:** Google Apps Script API, ExcelJS

## 📦 Установка
1. Клонировать репозиторий.
2. Выполнить `npm install`.
3. Создать файл `.env` (см. раздел Безопасность) и добавить токен бота.
4. Запустить: `node index.js`.
